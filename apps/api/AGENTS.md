# apps/api — Agent rules

Chi + Huma REST API. OpenAPI 3.1 spec auto-generated by Huma.

## Internal structure

```
internal/
├── config/       config.Load() reads env vars, returns *Config
├── middleware/   Auth() soft JWT middleware
├── handler/      HTTP only — no business logic
└── service/      All business logic
```

## Strict layering rule

**Handlers call services. Services call the DB. Never skip layers.**

```go
// ✅ Correct
func (h *Handler) getProject(ctx context.Context, input *GetProjectInput) (*GetProjectOutput, error) {
    proj, err := h.svc.Projects.Get(ctx, input.ID)
    // ...
}

// ❌ Wrong — never query GORM directly from a handler
func (h *Handler) getProject(...) {
    h.db.First(&proj, id)
}
```

## Huma patterns

Register operations with `huma.Register`, not router methods directly:

```go
huma.Register(api, huma.Operation{
    Method:      http.MethodGet,
    Path:        "/api/v1/projects/{id}",
    Summary:     "Get project",
    Security:    []map[string][]string{{"bearer": {}}},
    Tags:        []string{"Projects"},
}, h.getProject)
```

Error responses — always use Huma helpers, never raw JSON:

```go
return nil, huma.Error404NotFound("project not found", err)
return nil, huma.Error403Forbidden("not a member of this org", nil)
return nil, huma.Error400BadRequest("invalid slug format", nil)
```

## Authentication

`middleware.Auth()` is a **soft** middleware — it validates the JWT and sets the user ID in context if valid, but does NOT block unauthenticated requests. Handlers that require auth must call `requireUser(ctx)` explicitly:

```go
func requireUser(ctx context.Context) (uuid.UUID, error) {
    id, ok := middleware.UserFromContext(ctx)
    if !ok {
        return uuid.Nil, huma.Error401Unauthorized("authentication required", nil)
    }
    return id, nil
}
```

## Service layer conventions

- Each service struct holds only `db *gorm.DB`.
- Transaction-heavy operations (e.g., register = create user + create org + create membership) must use `db.Transaction()`.
- Return GORM errors unwrapped — the handler layer calls `notFound(err)` to convert `gorm.ErrRecordNotFound` → `huma.Error404`.

## Adding a new resource

1. Add model to `packages/db/models.go` + register in `db.Migrate()`.
2. Create `internal/service/<resource>.go` with CRUD methods.
3. Add `*<Resource>Service` field to `Services` struct in `service.go` and initialize in `New()`.
4. Create `internal/handler/<resource>.go` with Huma operations.
5. Add `h.register<Resource>Routes(api)` call in `handler.go`.

## Go workspace

This module resolves `packages/db` via a `replace` directive in `go.mod` — not from a remote registry. Do not change this to a remote import or pseudo-version.
